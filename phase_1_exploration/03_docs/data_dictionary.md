# Data Dictionary – Shopee Product

product_name
Nama produk sesuai tampilan Shopee

price
Harga produk dalam Rupiah (belum dibersihkan)

sold
Jumlah produk terjual (format teks Shopee)

rating
Nilai rating produk (0–5)

shop_name
Nama toko penjual

scrape_date
Tanggal data diambil

source
URL halaman produk
